プライバシーポリシー

最終更新日: 2024年12月1日

Chrome拡張機能「Drag search©」（以下、「本拡張機能」）におけるユーザー情報の取り扱いについて、以下の通りプライバシーポリシー（以下、「本ポリシー」）を定めます。

1. データの収集と利用について

本拡張機能は、ユーザーの個人情報、閲覧履歴、入力データを開発者のサーバーへ送信・保存することは一切ありません。

本拡張機能が行うデータ処理は、すべてユーザーのブラウザ内（ローカル環境）、またはユーザーが意図して操作した検索エンジンおよびWebサービスへの直接的な転送のみに限定されます。

2. 権限の使用目的

本拡張機能は、機能を提供するために以下の権限を使用します。これらの権限は記載された目的以外には使用されません。

  • ストレージ (storage): ユーザーの設定（ドラッグ方向ごとのアクション割り当て等）をブラウザ内に保存するために使用します。
  • タブ (tabs): 検索結果や指定されたWebサービスを新しいタブで開くために使用します。
  • スクリプティング (scripting): ユーザーがAIサービス（ChatGPT等）を選択した場合にのみ、そのページの入力欄にテキストを自動入力するために使用します。
  • クリップボード (clipboardWrite): ユーザーが「コピー」機能を選択した場合にのみ、テキストをクリップボードに書き込むために使用します。
  • 全サイトへのアクセス (Host Permissions): 閲覧中のWebページ上でドラッグ＆ドロップ操作を検知し、機能を提供するために必要です。データの読み取りはユーザーが選択したテキストに対してのみ行われます。

3. 外部サービスと免責事項

本拡張機能を使用することで、Google、Twitter、YouTube、各種AIサービス（ChatGPT, Claude, Gemini等）などの外部サービスにアクセスする場合があります。

  • これらの外部サービスにおけるデータの取り扱いは、各サービスのプライバシーポリシーに従います。
  • 本拡張機能はこれらのサービスの非公式ツールであり、各サービスの運営会社とは関係ありません。
  • 各サービスの仕様変更により、自動入力などの機能が予告なく利用できなくなる場合がありますが、開発者はその修正義務を負いません。

4. 収益化とアフィリエイトについて

本拡張機能は現在無料で提供されていますが、将来的な開発・維持費用のために、以下の施策を行う権利を留保します。

  • 検索結果や設定画面における広告の表示。
  • Amazonアソシエイト・プログラムや楽天アフィリエイトなどのアフィリエイトプログラムへの参加。

これらの施策が実施される場合でも、個人を特定するデータの収集や、ユーザーの意図しない不当なトラッキングは行いません。

5. 免責事項

本拡張機能の使用によって生じた、いかなる損害（データの損失、業務の中断、精神的損害等を含む）についても、開発者は一切の責任を負わないものとします。ユーザーは自己責任において本拡張機能を使用するものとします。

6. プライバシーポリシーの変更

開発者は、必要に応じて本ポリシーを変更することがあります。変更内容は本ページに掲載された時点から効力を生じるものとします。

7. お問い合わせ

本ポリシーに関するお問い合わせは、Chromeウェブストアのサポートページよりご連絡ください。

Privacy Policy

Last Updated: December 1, 2024

This Privacy Policy describes how "Drag search©" (the "Extension") handles user information.

1. Data Collection and Usage

The Extension does NOT collect, store, or transmit any personal information, browsing history, or input data to the developer's servers.

All data processing is performed locally within the user's browser or involves the direct transfer of text to the search engines or web services explicitly selected by the user.

2. Use of Permissions

The Extension uses the following permissions solely for the purpose of providing its functionality:

  • Storage: To save user preferences and settings locally.
  • Tabs: To open search results or web services in new tabs.
  • Scripting: To automatically insert text into input fields only when the user selects an AI service (e.g., ChatGPT).
  • ClipboardWrite: To copy selected text to the clipboard when the "Copy" action is triggered.
  • Host Permissions (All URLs): To detect drag-and-drop actions on web pages. We only access the text explicitly selected by the user.

3. External Services and Disclaimer

The Extension may interact with third-party services such as Google, Twitter, YouTube, and AI platforms (ChatGPT, Claude, Gemini, etc.).

  • Data handling on these external services is subject to their respective privacy policies.
  • This Extension is an unofficial tool and is not affiliated with these service providers.
  • The developer is not responsible if features (e.g., autofill) stop working due to changes in third-party services.

  • Display advertisements within the search results or settings page.
  • Participate in affiliate programs (e.g., Amazon Associates, Rakuten Affiliate).

Even if these measures are implemented, we will not collect personally identifiable information or engage in unauthorized tracking.

5. Limitation of Liability
